Migration plan

  • Pick an icon set: Candidates are Lucide (MIT, 1500+ icons, 24px grid, clean stroke style), Heroicons (MIT, by Tailwind team, 24px+20px, outline/solid), Phosphor (MIT, 6 weights, 256px viewBox), and Tabler Icons (MIT, 5700+, 24px, stroke-based).
  • Requirements: MIT or ISC licence. Stroke-based (matches our existing SVGs). Good rendering at 12–16px (toolbar icon size). Includes: x/close, search, filter, chevron, external-link, copy, eye-off, star, pencil, arrow-left/right, undo, plus, heart.
  • Priority order: (1) Delete/close × — 7+ uses, fragile centering. (2) Star ★ — rendering varies across OS. (3) Pencil ✎ — ditto. (4) Arrows & undo. (5) Decorative (heart, separators).
  • Implementation: Inline SVG strings in render_html.py (same as current toolbar icons). No icon font, no sprite sheet, no external files — reports must be self-contained single-file HTML.
  • Existing SVGs: Our 7 custom SVGs use varied viewBox sizes (10, 12, 16, 24) and stroke widths (1.5, 2). The chosen icon set should normalise these to a consistent grid.
