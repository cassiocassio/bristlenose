One user-facing word per axis, identical across CLI · env · SPA · Mac app. "theme" is banished from every user surface (it means research findings — the Themes tab); internal data-color-theme/color_scheme stay as-is. CLI & env are English-only; the SPA & Mac columns re-render in the locale you pick.
Locale

⚠ Why "palette" and never "theme" — the CJK collision

In Latin-script locales, appearance / palette / theme are three unrelated everyday words, so there's no clash. But in Japanese, Korean, and (proposed) Traditional Chinese, the software word for a colour "theme" is the same loanword already shipped for research findings — so a "colour theme" picker would read identically to the Themes report tab. The escape is the "palette" word:

ja — research テーマ  =  software "theme" テーマ  →  palette must be カラーパレット (or 配色)
ko — research 테마  =  software "theme" 테마  →  palette must be 색상 팔레트
zh-Hant — research 主題  =  software "theme" 主題  →  palette must be 調色盤 (or 配色). Never 主題 / 佈景主題.

Current → proposed env/CLI (migration). Today only env vars exist, and they fight the UI: BRISTLENOSE_COLOR_SCHEME (light/dark) vs BRISTLENOSE_COLOR_THEME (palette) are near-homophones for different axes, and "theme" is the banished word. Proposal renames the user-typed surfaces to --appearance/BRISTLENOSE_APPEARANCE, --palette/BRISTLENOSE_PALETTE, --typography/BRISTLENOSE_TYPOGRAPHY; keep old env names as deprecated aliases for the desktop Swift sidecar (grep desktop/ before renaming). No --color-theme CLI flag exists today.

"Edo" & "Default". "Edo" is a proper noun → keep as a name: Latin "Edo" in en/es/fr/de/cs/pt; native script for CJK (ja/zh-Hant 江戸, ko 에도) — a light branding call, flag for native reviewers. "Default" is ordinary chrome → translate via each platform's Apple term (es Predeterminada, pt-PT Predefinida, ko 기본값, zh-Hant 預設).

Appearance is already Apple-aligned in every shipped locale (Erscheinungsbild / 外観 / 외관 / Vzhled / Apparence …) — leave axis 1 untouched. zh-Hant would use macOS 外觀. Locales: 9 shipping (en, es, fr, de, ko, ja, cs, pt-BR, pt-PT) + zh-Hant to come.
