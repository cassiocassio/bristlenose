Signal grid — sizing sandbox

The person badge (sized for a two-digit code + padding) is the unit: it sets the square cell, the max circle, and the vertical rhythm (row height = cell). Step the cell size down until a typical section/theme count fits a reasonable vertical space — fonts never change, only the boxes. Watch the tallest-pane height in the readout.

—

Cell size and gutter are the only sliders and they touch geometry only — every font stays fixed. The badge's horizontal padding shrinks with the cell (clamp()) so its fixed-size text is the natural floor. Dot max diameter = cell − 10px; area (never radius) ∝ residual.
