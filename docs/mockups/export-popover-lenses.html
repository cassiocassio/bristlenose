Export — one popover, five lenses

The toolbar Export ↑ popover keeps a fixed frame — Anonymise + Export Report are on every tab — and morphs only the middle block to fit the lens you're on. Static approximation of the SwiftUI; real build differs slightly.

Decisions baked in
  • Project / Analysis / Codebook → just Export Report for TF (the dashboard is the whole report; analysis signal-cards ride the self-contained HTML; codebook export is post-TF).
  • Sessions → Reveal Transcripts in Finder — the transcripts are already files on disk; the Mac just opens the folder (local-first). CLI users are already there.
  • Quotes → Copy / Spreadsheet / Clips. Copy Quotes is a plain action by default and grows a scope picker only when you have a selection or stars (Option B).
  • Anonymise is global (strips participant names). It does not apply to Reveal-in-Finder (that opens the real on-disk files) — an anonymised transcript bundle is a separate, post-TF export.
