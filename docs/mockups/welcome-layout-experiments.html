Welcome screen — layout experiments

Wireframe only — no styling or hierarchy intent yet. The same fixed content set is poured into a dozen page divisions from grid theory, so we can feel word balance and whitespace before choosing a direction. Non-scrolling WelcomeView (SwiftUI), ~80 words of content, ≤6 tweets.

Content atoms in every frame: ◐ icon · Header (title + 22-word line) · How it works (3 steps) · New project · Set up your AI · Tip (dismissible, links out).
