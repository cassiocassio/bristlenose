Live in the report

The report is where the work happens, not just where it ends up. Star quotes, hide noise, tag with your codebook, edit participant names, rearrange the codebook. Search any quote across every interview. Changes persist between sessions.

AutoCode proposes tags from the AI; you set a confidence threshold and review the borderline cases.

Closing the gap between interviews and findings

You've run your interviews. The recordings are in a folder and stakeholders want findings.

Point Bristlenose at your recordings — audio, video, or transcripts from Zoom, Teams, or Google Meet — and it handles the mechanical part. Transcription runs on your laptop. The analysis pass extracts quotes, identifies speakers, groups everything by screen and theme. A typical run takes two to five minutes.

Built by a practising user researcher, not a platform company. Free, open source, no accounts, no telemetry.

Try it now

(Mac, command line)

brew install cassiocassio/bristlenose/bristlenose
bristlenose ~/Desktop/interviews/

Opens the report in your browser when it's done.

Get Homebrew first if you need it

Your research project, on your laptop.

The project file lives on your machine. You pick which AI provider sees the transcripts — Claude, ChatGPT, Gemini, Azure, or none of them (Ollama runs the analysis locally). No Bristlenose server, no account, no telemetry.

API keys live in your OS keychain, never in plaintext. Optional PII redaction strips names, phone numbers, and ID numbers from transcripts before analysis, with an audit trail.

Costs

A typical study costs about $1.50 in API fees. You bring your own API key, or free from the command line with a local model.
